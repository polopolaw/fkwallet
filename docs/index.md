## Overview

The FKWallet package provides a complete integration with the fkwallet.io API, allowing you to:

- Get wallet balances
- View transaction history
- Create withdrawals and transfers
- Manage online product orders
- Access payment systems and currencies

## Features

- Full API coverage (14 methods)
- Type-safe DTOs and Value Objects
- Automatic signature generation
- Retry mechanism for failed requests
- Comprehensive error handling
- Laravel Facade support
- **Proxy support** - Use proxy servers for requests
- PSR-12 compliant code
- Full test coverage
